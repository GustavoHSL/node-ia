# 🤖 node-ia

Projeto de estudo em **Node.js** e **TypeScript**, desenvolvido a partir de aulas da Rocketseat, com adaptações para a API atual da OpenAI.

---

## 🚀 Tecnologias

* Node.js
* TypeScript
* OpenAI SDK
* dotenv
* tsx

---

## 📚 Sobre

Este repositório tem como objetivo praticar:

* fundamentos de backend com Node.js
* uso de TypeScript no dia a dia
* consumo de APIs externas
* integração com inteligência artificial

---

## ⚙️ Setup

```bash
npm install
```

Crie um arquivo `.env`:

```env
OPENAI_API_KEY=your_api_key_here
```

---

## ▶️ Executar

```bash
npx tsx src/index.ts
```

---

Projeto com foco em aprendizado. Algumas decisões e implementações podem ser simplificadas propositalmente.
